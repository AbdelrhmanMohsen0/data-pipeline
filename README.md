# Data Processing Pipeline
A console app to process datasets, implemented twice using functional and imperative programming paradigms.

## Setup
- Install [uv](https://github.com/astral-sh/uv)
    ```sh
    pip install uv
    ```
- Clone the repository
    ```sh
    git clone https://github.com/AbdelrhmanMohsen0/data-pipeline.git
    ```
    ```sh
    cd data-pipeline
    ```
- Install required packages
    ```sh
    uv sync
    ```
## Usage
```sh
uv run main.py ./data/sales_data.csv
```
